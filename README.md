# 🏛️ Government Fraud Analytics Platform

## 🚀 Enterprise AI-Powered Fraud Detection & Budget Intelligence System

An enterprise-scale AI-driven Fraud Detection and Budget Analytics Platform designed for government financial systems, public expenditure monitoring, anomaly detection, and real-time risk intelligence.

---

# 🌐 Business Problem

Government departments process massive amounts of:

- Financial transactions
- Budget allocations
- Vendor payments
- Public procurement records
- Tax expenditure reports
- Department spending activities

Traditional systems struggle to:

- Detect anomalies in real-time
- Identify suspicious transactions
- Monitor budget overutilization
- Detect fraud patterns proactively
- Provide centralized risk intelligence dashboards

This platform solves those problems using AI-powered analytics.

---

# 🎯 Project Objectives

✅ Detect high-risk financial anomalies

# 🚀 Running the Project

## Install Dependencies

```bash
pip install -r requirements.txt
```

## Run Backend

```bash
uvicorn backend.api:app --reload
```

## Open Swagger Docs

```text
http://127.0.0.1:8000/docs
```

## Launch Frontend

Open:

```text
frontend/index.html
```
